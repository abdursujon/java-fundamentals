# Java Fundamentals

A practical way to learn Java fundamentals. 
Every topic is a small, self-contained, runnable `.java` file you can read, edit, and execute. 
This project is simple and made beginner-friendly so anyone can use it as a guide to start learning java.
It also provides with comments whenever necessary to clarify implementation for beginners. 
The project also practiced simplified testing procedure through using main method on each class  
to show example output which you can run by IDE run button or commands as stated below. 

## Example run 

```bash
cd fundamentals/chapter1_basics
java Variables.java
```
